# JavaScript Course Projects

Цей репозиторій містить практичні завдання з курсу **JavaScript GoIT**.  
Кожен модуль — це окремий етап розвитку інтерактивних веб-застосунків.

---

## Структура

- `module-01-html-basics` — основи HTML  
- `module-02-css-basics` — основи CSS для JS-проєктів  
- `module-03-js-basics` — синтаксис JS, змінні, оператори  
- `module-04-js-functions` — функції, об’єкти, масиви  
- `module-05-js-dom` — робота з DOM, події, класи  
- `module-06-js-forms` — форми, валідація, події форм  
- `module-07-js-arrays-objects` — масиви та об’єкти, методи об’єктів  
- `module-08-js-events-timers` — таймери, setTimeout, setInterval, події  
- `module-09-js-ajax` — fetch, HTTP-запити, робота з API  
- `module-10-js-timer-promises` — таймер зворотного відліку, проміси, flatpickr, iziToast  
- `module-11-js-axios-gallery` — галерея з Axios, SimpleLightbox, css-loader, iziToast  
- `module-12-js-async-gallery` — асинхронні запити з async/await, пагінація, кнопка Load more

---

## Технології

- **HTML5** — структура сторінок  
- **CSS3** — стилі та адаптивність  
- **JavaScript (ES6+)** — логіка та асинхронні запити  
- **Vite** — збірка проєктів  
- **Axios** — HTTP-запити  
- **Flatpickr** — вибір дати та часу  
- **iziToast** — повідомлення користувачу  
- **SimpleLightbox** — модальні вікна для галереї  
- **css-loader** — індикатор завантаження

## Фінальні проєкти

Кожен модуль містить свій проєкт із готовою структурою та функціоналом.
